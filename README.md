## Usage
- Play a Song
  - ```!p [Song Name/Link]```
- Pause Currently Playing Song
  - ```!pause```
- Skip Currently Playing Song
  - ```!skip [Song Name/Link]```
- Leave Voice Channel
  - ```!p [Song Name/Link]```
